# GameStore.API

## Basic Instructions

- build: Run `dotnet build` in the GameStore.Api dir terminal
- run: Press F5 in `Program.cs`
- run with debugger: In Solution Explorer, right click the GameStore.Api and select `Debug` > `Start New Instance`
- run without debugger: In Solution Explorer, right click the GameStore.Api and select `Debug` > `Start without Debugging`
- run from the terminal: `dotnet run` in GameStore.Api dir
